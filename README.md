# final_thesis_data
This repository includes the full raw dataset for my thesis:  Hidden in Plain Sight: Model Minority Stereotypes and the Suppression of South Asian Health Risk. 
# Appendix Q – Complete Study Data

This repository contains the full raw dataset used in this study, corresponding to Appendix Q.

## Contents

The dataset is organized into 10 sheets:

- **H1 - TIME TRENDS**: Data supporting temporal trends analyzed for H1.  
- **H1 - FINAL FOR ANALYSIS**: Cleaned dataset used for final analyses for H1.  
- **H1 - MASTER**: Complete raw dataset for H1, including studies that were ultimately excluded.  
- **H1 - CVD MANUAL CHECK**: Subset of samples manually coded for CVD focus.  

- **H2 - EPI BLACK**: Studies analyzed for first epidemiological mention for Black Americans.  
- **H2 - EPI SA**: Studies analyzed for first epidemiological mention for South Asians.  
- **H2 - INST BLACK**: Studies analyzed for first institutional recognition for Black Americans.  
- **H2 - INST SA**: Studies analyzed for first institutional recognition for South Asians.  

- **H3 - FINAL**: Documents analyzed and coded for H3.  
- **H3 - SUMMARY TABLE**: Summary table of key findings for H3.  

## Notes

- The dataset includes all raw and processed data used in the study.  
- Variable names and coding schemes are consistent across sheets.  
- Due to the size of the dataset, it is provided as a separate file rather than embedded in the main manuscript.  
- For H2 sheets, **dark green highlighted rows** indicate the final studies identified as initial recognition.  
- In **H2 - EPI BLACK**, an additional row is highlighted in **light green**. This represents the first study that satisfied inclusion criteria; however, it did not meet the citation threshold required to be considered a landmark study.
- For H2 EPI sheets, any study flagged as ‘False’ for a given criterion was excluded from consideration for subsequent criteria.
